# FreeMoCap Web

Register: product

FreeMoCap Web is a browser front end for the FreeMoCap video-import workflow. It is for researchers, educators, and creators who already have recorded `.mp4` camera videos and want to place them into the standard FreeMoCap recording folder structure without launching the desktop Qt UI.

The product deliberately does not include live camera recording. Processing, tracking, synchronization, status checks, and recording-folder conventions must come from the upstream FreeMoCap Python package and its dependencies rather than local reimplementations.

The interface should feel like a compact lab tool: direct, calm, file-system aware, and explicit about what is happening to recordings.
